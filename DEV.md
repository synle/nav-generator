# nav-generator

Web application to organize and generate bookmarks / navigation pages. Built with React 19, Vite, SCSS, and Monaco Editor; tested with Vitest.

## Quick Start

Install dependencies:

```bash
npm ci || npm install --no-fund --prefer-offline
```

Run the dev server (watches and serves via `npm start`):

```bash
./dev.sh
```

Run tests:

```bash
npm test
```

Build for production:

```bash
npm run build
```
